# 设置

设置页面是你控制 Whale Play 外观、行为以及连接到 AI 服务的地方。本指南将带你了解每个部分。

---

## 如何打开设置

点击左侧边栏中的**齿轮图标** ⚙️ 打开设置页面。

---

## API 配置

此部分用于将 Whale Play 连接到 AI 服务。使用该应用需要 API Key。

### 支持的提供商

| 提供商                 | 你需要什么                                                                                          |
| ---------------------- | --------------------------------------------------------------------------------------------------- |
| **DeepSeek**           | 来自 [platform.deepseek.com](https://platform.deepseek.com/) 的 API Key                             |
| **兼容 OpenAI 的服务** | 任何支持 OpenAI API 格式的提供商（OpenAI、Groq、Together 等）。你需要一个 Base URL 和一个 API Key。 |

### 字段

| 字段         | 作用                                                                                                            |
| ------------ | --------------------------------------------------------------------------------------------------------------- |
| **API Key**  | 你用于 AI 提供商的密钥。请保密——不要分享。                                                                      |
| **Base URL** | AI API 的服务器地址。对于 DeepSeek，这是 `https://api.deepseek.com`。对于其他提供商，请使用其 API 端点。        |
| **Model**    | 使用哪个 AI 模型。不同模型有不同的优势（速度、创造力、推理能力）。常见选项：`deepseek-chat`、`gpt-4o-mini` 等。 |

![设置 API 页面](../images/settings-api-config.png)

## Temperature、Max Tokens 与 Reasoning Effort

这些设置控制 AI 的思考和写作方式。

| 设置                 | 作用                                                                                     | 较低值               | 较高值                |
| -------------------- | ---------------------------------------------------------------------------------------- | -------------------- | --------------------- |
| **Temperature**      | 控制随机性。较低 = 更可预测、更专注的回复。较高 = 更具创意、更出人意料的回复。           | 0.3 — 安全、重复性高 | 1.2 — 狂野、有创意    |
| **Max Tokens**       | 每条 AI 回复的最大长度。一个 Token 大约相当于一个英文单词。512 tokens ≈ 350 个单词。     | 256 — 简短回复       | 2048 — 长篇独白       |
| **Reasoning Effort** | （仅限支持的模型）模型在回答前"思考"的程度。越高的 effort 意味着更好的推理但更慢的响应。 | Low — 快速回复       | High — 深思熟虑的回答 |

### 推荐起始值

| 使用场景             | Temperature | Max Tokens |
| -------------------- | ----------- | ---------- |
| 快速随意聊天         | 0.7         | 512        |
| 创意故事叙述         | 0.9         | 1024       |
| 详细世界构建         | 0.8         | 1536       |
| 快速、直奔主题的回复 | 0.4         | 256        |

## 上下文 Token

此设置控制 AI 可以"记住"多少聊天历史。

| 设置                    | 作用                                                                                       |
| ----------------------- | ------------------------------------------------------------------------------------------ |
| **Context Token Limit** | AI 在生成回复时考虑的最大 Token 数量。这包括角色卡片、人设、活跃的世界书条目以及聊天历史。 |

### 工作原理

- 如果你的聊天历史 + 角色卡片 + 世界书条目 = 10,000 tokens，而上下文限制为 8,192，Whale Play 会**首先修剪最旧的消息**以保持在限制内。
- 更大的上下文意味着 AI 能记住更多对话内容，但也会消耗更多 Token（如果你按 Token 计费，可能会花费更多）。

### 建议

| Context Limit | 使用场景                                               |
| ------------- | ------------------------------------------------------ |
| 4,096 (4K)    | 简短、专注的对话。消耗较少 Token。                     |
| 8,192 (8K)    | 大多数聊天的良好默认值。                               |
| 16,384 (16K)  | 长篇幅、细节丰富的角色扮演会话。                       |
| 32,768 (32K)  | 包含许多世界书条目的超长故事。仅在你的模型支持时使用。 |

## 正则表达式规则

正则表达式规则让你**自动清理或修改 AI 的回复**，然后再展示给你。这是用于微调输出的高级功能。

### 你能做什么

| 规则类型             | 作用                                 | 示例                                                   |
| -------------------- | ------------------------------------ | ------------------------------------------------------ |
| **Strip**            | 从 AI 回复中移除不需要的文本。       | 移除像 `**bold**` 或 `*italic*` 这样的 Markdown 格式。 |
| **Template**         | 将特定模式替换为自定义文本。         | 将 `{{char}}` 替换为角色名称。                         |
| **Display Template** | 改变文本的显示方式，不影响底层内容。 | 在显示时为对话加上引号。                               |

### 常见用途

- 移除多余空格或空行
- 将 `{{user}}` 占位符替换为用户的人设名称
- 移除用于斜体格式的星号
- 移除 "System:" 或 "Assistant:" 前缀

> **提示：** 正则表达式是一种模式匹配语言。如果你不熟悉它，可以从简单的模式开始，比如将一个词替换为另一个词。

完整的可用正则规则类型和语法参考，请查看[正则规则参考](../reference/regex-rules.md)。

## 语言选择

Whale Play 的界面支持多种语言。

| 语言    | 标签    |
| ------- | ------- |
| English | English |
| Chinese | 中文    |

更改语言的方法：

1. 进入设置。
2. 找到 **Language** 下拉框。
3. 选择你偏好的语言。
4. 界面会立即更新。

> **注意：** 语言选择仅影响应用界面（按钮、标签、菜单）。它不控制 AI 回复的语言。要影响 AI 的语言，请在角色卡片或预设中说明。
